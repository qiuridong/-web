| 教训 | 适用场景 |

| **GitHub gh api search 优于 webfetch** | 本会话 webfetch GitHub 一直碰认证墙,gh api search/contents 直接拿数据,且支持 base64 解码 |
| **建议方案前先把"现有方案的失败模式"挖深** | 我先做了 5 方向调研才发现"原脚本 CF 33/33 100%",所以前 4 方向的"过 CF"投入都不必要;**应该先归因再调研** |
