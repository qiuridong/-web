## 期间修的 bug

1. **`backend/app/services/settings_service.py:set()`** 函数名遮蔽 builtin `set` → 导致 `ensure_defaults` 里 `set[str]` 类型注解的 `set(...)` 调用报 "missing args" 错(6B agent 自修;`set_value` 主名 + `set = set_value` 别名 + `ensure_defaults` 用 `builtins.set`)
2. **`scheduler.trigger_now`** 一开始用 `asyncio.get_event_loop()` 在 sync FastAPI 路由(worker thread)抛 `RuntimeError: There is no current event loop` → 改为读 APScheduler 内部 `_eventloop` + 用 `run_coroutine_threadsafe` 跨线程提交(6A agent 自修)
3. **`frontend/src/hooks/use-mobile.tsx`** MOBILE_BREAKPOINT 从 shadcn 默认 768 改为 480 — 桌面浏览器窗口缩小到 480-1280px 不再被误判为 mobile(原行为是 Sheet overlay 覆盖主内容)
