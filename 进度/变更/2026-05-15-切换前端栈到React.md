# 2026-05-15 · 切换前端栈到 React

## 起因

用户明确指示:**"换 React,UI 调用美化的技能或相关技能继续工作,如果没有自行安装"**。

## 美化"技能"评估结果

按用户要求先调研可用的 UI 美化 MCP / 技能:

| 渠道 | 结果 |
|---|---|
| `mcp__mcp-registry__search_mcp_registry` 搜 `shadcn / ui / react / tailwind / design / components / magic / 21st / frontend / css / figma / browser / screenshot / preview` | 全部空结果,**无可装 MCP** |
| 现有已连接 connectors | 同样无相关项 |
| 系统 deferred 工具列表 | 发现 **`Claude Preview`** 系列(`preview_start` / `preview_screenshot` / `preview_eval` / `preview_inspect` / `preview_console_logs` / `preview_network` 等)— 这就是后续编码阶段用来实时验证 UI 效果的杀手锏 |
| 现有 anthropic-skills | `skill-creator` 可创建 skill,但本任务不必新造 |

**结论**:无须额外安装 MCP。`Claude Preview` 已是当下最适合"前端 UI 美化迭代"的工具(跑 dev server → 截图 → 改样式 → 再截图对比)。后续编码 agent 可用它在浏览器实时看 UI 效果。

## 技术栈对比(变更前 → 变更后)

| 层 | 旧 Vue 栈(已废弃) | 新 React 栈 |
|---|---|---|
| 框架 | Vue 3 + Composition API | **React 18 + TypeScript** |
| 构建 | Vite + Pinia + Vue Router 4 | Vite + **React Router 7** |
| 状态 | Pinia | **Zustand**(全局) + **TanStack Query 5**(服务端数据) |
| UI 库 | Element Plus(重度美化) | **shadcn/ui**(Radix + Tailwind CSS v4) |
| 样式 | SCSS + Element Plus theme tokens | **Tailwind v4** + CSS variables (`@theme` directive) |
| 图标 | unplugin-icons + iconify | **lucide-react**(shadcn 默认) |
| 表单 | Element Plus Form | **react-hook-form + zod** |
| 数据可视化 | echarts(规划) | **Recharts** + **Tremor v3**(KPI 借鉴) |
| 动画 | @vueuse/motion | **Framer Motion 12** |
| 命令面板 | 自写 | **cmdk** |
| Toast | Element Plus Message | **sonner** |
| 主题切换 | useDark (VueUse) | **next-themes** |
| HTTP 类型 | openapi-typescript + openapi-fetch | 同(沿用) |
| SSE | EventSource(原生) | **@microsoft/fetch-event-source**(支持自定义 header + 自动重连) |
| 终端日志 | 自写 LogViewer | **xterm.js (@xterm/xterm)** |
| Markdown | 待定 | **react-markdown + remark-gfm** |
| 测试 | Vitest + Vue Test Utils | **Vitest + React Testing Library** |

## 设计语言保留

视觉设计(色板、字号、间距、圆角、阴影、动画 token)的**设计原则不变**(见 ADR-011 末尾说明):
- 调性五词:**克制 / 现代 / 精致 / 安静 / 可信赖**
- 主色 Indigo,强调色 Teal,语义色四档,中性色暖灰
- 深色模式一等公民,字体 Inter Variable + HarmonyOS Sans SC,数字 tabular-nums
- 圆角偏圆(8-12px 起步),阴影深色模式带 inset highlight
- 美化参考:Linear / Vercel / Raycast / cal.com / shadcn examples

只是**承载这些 token 的容器从 Element Plus CSS vars 换到了 shadcn/ui 的 OKLCH CSS vars**。
