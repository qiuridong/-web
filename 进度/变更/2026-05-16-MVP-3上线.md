### PM hotfix(本批关键防御)
- **Dashboard 4 处 `.map(...)` 防御性 `?? []` 保护**(防止后端字段缺失/loading 时 undefined.map 顶级崩):
  - `overview?.sparkline_7d_success.map` → `(overview?.sparkline_7d_success ?? []).map`

- **`<RouteErrorBoundary />` 全局兜底**(新建 `components/common/RouteErrorBoundary.tsx`):
  - 替换 React Router 默认的 "Unexpected Application Error!" 丑陋页
  - 友好降级:大警告图标 + 中文 "页面出错了" + 错误详情 collapsible + 3 按钮(刷新/返回/回首页)

  - 以后任何路由 throw 不再白屏 + 红字 stack
- **nginx `index.html` 加 `Cache-Control: no-cache, no-store, must-revalidate`**(根治"用户拿不到新 dist"问题):
  - 之前老 dist `index-DcfiIsJJ.js` 被浏览器缓存,用户报 dashboard / 改密页崩 — 真因是用的老版没有相应代码或字段
